# 拡張機能のセットアップ（Chromeへの読み込み方法）

## 1. 開発サーバーの起動
拡張機能を開発モードでビルド・監視するために、ターミナルで以下のコマンドを実行してください。

```bash
npm run dev
```

※ これにより `dist` フォルダが生成・更新されます。開発中は基本的にこのコマンドを実行したままにします。

## 2. Chromeでの読み込み手順

1. Chromeを開き、アドレスバーに以下を入力してエンターキーを押します。
   ```text
   chrome://extensions
   ```
2. 画面右上の **「デベロッパーモード」** スイッチを **ON** にします。
3. 左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
4. フォルダ選択ダイアログで、このプロジェクトの **`dist`** フォルダを選択します。
   - プロジェクトパス: `D:\Script\Chrome\my-extension\dist`

## 3. 確認
- 拡張機能の一覧に **「FluxMarks」** が追加されていれば成功です。
- ツールバーのパズルピースアイコン🧩をクリックして、FluxMarksをピン留めしておくと便利です。
- アイコンをクリックしてサイドパネルが開くことを確認してください。

## トラブルシューティング
- **変更が反映されない場合**:
  - `npm run dev` が動いているか確認してください。
  - 拡張機能管理画面（chrome://extensions）で、FluxMarksのカードにある **更新ボタン（回転矢印アイコン）** をクリックしてください。
