# FluxMarks

FluxMarksは、Chromeのサイドパネルでブックマークを効率的に管理するための拡張機能です。
フォルダの作成、ドラッグ＆ドロップによる並べ替え、コンテキストメニューからの操作など、ネイティブに近い操作感を提供します。

## 機能

- **サイドパネル表示**: ブラウジングを邪魔せずにブックマークにアクセス
- **ブックマーク挙動管理**: 新規タブ、バックグラウンドタブ、再読み込みなどをブックマークごとに管理できます。
- **フォルダ一括オープン**: フォルダ内のすべてのブックマークを一度に開くことができます。再帰的に下のフォルダまで開けますが、開く数は注意してください。
- **ドラッグ＆ドロップ**: 直感的な並べ替え
- **コンテキストメニュー**: 新しいフォルダの作成、編集、削除
- **高速な検索**: 膨大なブックマークからも瞬時に検索（予定）

## インストール方法（開発者向け）

1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/YourUsername/FluxMarks.git
   cd FluxMarks
   ```
2. 依存関係をインストールします。
   ```bash
   npm install
   ```
3. ビルドを実行します。
   ```bash
   npm run build
   ```
4. Chromeを開き、`chrome://extensions` にアクセスします。
5. 右上の「デベロッパーモード」をオンにします。
6. 「パッケージ化されていない拡張機能を読み込む」をクリックし、`dist` フォルダを選択します。

## 開発

### 開発サーバーの起動
```bash
npm run dev
```
HMR（ホットモジュールリプレースメント）が有効な状態で開発できます。
