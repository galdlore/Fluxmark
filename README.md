# FluxMarks

FluxMarksは、Chromeのサイドパネルでブックマークを効率的に管理するための拡張機能です。
フォルダの作成、ドラッグ＆ドロップによる並べ替え、コンテキストメニューからの操作など、ネイティブに近い操作感を提供します。

## 機能

- **サイドパネル表示**: ブラウジングを邪魔せずにブックマークにアクセスできます。
- **セッション保存**: 現在のウィンドウで開いているすべてのタブを、日時付きのフォルダとして一括保存できます（ヘッダーの💾アイコン）。
- **セーフティモード**: 誤操作を防ぐための「読み取り専用モード」を搭載（ヘッダーの🔓/🔒アイコンで切り替え）。
- **ブックマーク挙動管理**:
  - **グローバル設定**: ブックマークを開く際のデフォルト挙動（新しいタブ（背面/前面）、現在のタブ）をヘッダーから一括設定できます。
  - **個別設定**: コンテキストメニューから、特定のブックマークやフォルダごとに挙動を上書き設定できます。
- **ドラッグ＆ドロップ**: 直感的な操作でブックマークの並べ替えやフォルダ移動が可能です。フォルダの上にドラッグして少し待つと自動でフォルダが開く「ホバー展開」機能を搭載。
- **非表示 / 復元 (Hide & Restore)**: 削除の代わりに「非表示」にする機能を提供。誤って消してしまうことを防ぎます。非表示アイテムは「👁️」アイコンで表示/非表示を切り替えられ、コンテキストメニューから簡単に復元できます。
- **コンテキストメニュー**: フォルダ作成、名前変更、非表示(Hide)、復元(Restore)、挙動設定など、右クリックから多彩な操作が可能です。
- **堅牢なレイアウト**: ヘッダーを固定し、リスト部分のみがスクロールする快適な操作性を提供します。
- **ブックマークを汚染しない**: フォルダ移動や追加・削除はChrome標準のブックマークと同期しますが、**並び順**や**非表示状態**は拡張機能独自の「仮想ツリー」として管理されます。これにより、自分だけの使いやすい並び順を維持できます。
- **高速な検索**: インクリメンタルサーチで目的のブックマークを素早く見つけられます。
- 修正したこの機能拡張カスタムブックマークは保存/エクスポート/インポートはChromeの制限により、実装できませんでした。（デバイス/プロファイルごとにBookmark IDが変わってしまう可能性があるため個別ブックマーク識別ができない）

## インストール方法（開発者向け）

1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/YourUsername/FluxMarks.git
   cd FluxMarks
   ```
2. 依存関係をインストールします。
   ```bash
   npm install
   ```
3. ビルドを実行します。
   ```bash
   npm run build
   ```
4. Chromeを開き、`chrome://extensions` にアクセスします。
5. 右上の「デベロッパーモード」をオンにします。
6. 「パッケージ化されていない拡張機能を読み込む」をクリックし、`dist` フォルダを選択します。

## 開発

### 開発サーバーの起動

```bash
npm run dev
```

HMR（ホットモジュールリプレースメント）が有効な状態で開発できます。
